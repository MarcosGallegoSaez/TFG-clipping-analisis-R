# TFG - Análisis estadístico sobre content clipping

Este repositorio contiene el código y los archivos de análisis utilizados en el Trabajo de Fin de Grado:

**Análisis del content clipping como modelo emergente de visibilidad orgánica en marketing digital**

Autor: Marcos Gallego Sáez  
Universidad: CUNEF Universidad  
Curso: 2025-2026

## Contenido del repositorio

- `codigo_analisis_tfg_clipping.R`: script principal de análisis estadístico en R.
- `datos_encuesta_anonimizados.csv`: base de datos anonimizada exportada desde Google Forms.
- `tablas_resultados/`: tablas generadas durante el análisis.
- `figuras_resultados/`: gráficos generados durante el análisis.

## Descripción del análisis

El análisis estadístico incluye:

- análisis descriptivo de la muestra;
- descriptivos de variables medidas mediante escala Likert;
- recodificación de categorías para facilitar la interpretación;
- pruebas KMO y Bartlett;
- análisis factorial exploratorio;
- análisis de fiabilidad mediante alfa de Cronbach;
- correlaciones de Spearman;
- contraste binomial;
- regresión lineal múltiple.

## Nota sobre los datos

Los datos incluidos en este repositorio están anonimizados y no contienen información personal identificativa de los participantes.

## Software utilizado

El análisis se realizó en R.
